## Requirements
+ socket

## Setup
```
pip install socket

git clone https://github.com/Cod3zghost/RAT-Python3

cd RAT-Python3

chmod +x server.py

chmod +x client.py

nano server.py (edit to your custom lhost and lport)

nano client.py (edit to your custom rhost and rport)

```

## Usage (Server)
```
python3 server.py
```

## Usage (Client)
```
python3 client.py
```
